# VirusTotal API Client

Простой Python‑скрипт для получения статуса сканирования файла через VirusTotal API по его хешу (MD5/SHA1/SHA256).

## Возможности

* Проверка файлов по хешу через VirusTotal API.
* Вывод основных результатов сканирования в консоль.
* Сохранение полного JSON‑ответа от API в файл.
* Обработка ошибок соединения и ответов API.

## Требования

* Python 3.6+
* Библиотека `requests` (установите через `pip install requests`)
* API‑ключ от VirusTotal (получите на [virustotal.com](https://www.virustotal.com))

## Установка и настройка
Инструкция по запуску скрипта
Шаг 1. Подготовка окружения
Убедитесь, что у вас установлен Python (версия 3.6 или выше). Проверить версию можно командой:

bash
python --version
Установите необходимую библиотеку:

bash
pip install requests
Библиотека json входит в стандартную библиотеку Python, её устанавливать не нужно.

Шаг 2. Настройка API‑ключа
Установите API‑ключ VirusTotal в переменную окружения VT_API_KEY:

Windows (командная строка):

cmd
set VT_API_KEY="Ваш API ключ"
Windows (PowerShell):

powershell
$env:VT_API_KEY=""Ваш API ключ""
Linux/macOS:

bash
export VT_API_KEY=""Ваш API ключ""
Чтобы переменная сохранялась между сессиями:

В Linux/macOS добавьте команду export в файл ~/.bashrc или ~/.zshrc.

В Windows настройте переменную через Панель управления → Система → Дополнительные параметры системы → Переменные среды.

Шаг 3. Запуск скрипта
Перейдите в директорию со скриптом:

bash
cd /путь/к/скрипту
Запустите скрипт:

bash
python virustotal_api.py
При запросе введите хеш файла для проверки (поддерживаются форматы MD5, SHA1, SHA256).

Шаг 4. Анализ результатов
После выполнения запроса скрипт:

выведет основные результаты сканирования в консоль;

сохранит полный JSON‑ответ от API в файл virustotal_response.json.
Установите зависимости:

bash
pip install -r requirements.txt
Настройте API‑ключ:

Создайте файл .env в корне проекта.

Добавьте в него строку:

VT_API_KEY=ваш_api_ключ_здесь
Либо установите переменную окружения (см. ниже).

Использование
Установите API‑ключ в переменную окружения:

Linux/macOS:

bash
export VT_API_KEY="ваш_api_ключ"
Windows:

cmd
set VT_API_KEY=ваш_api_ключ
Запустите скрипт:

bash
python virustotal_api.py
Введите хеш файла (MD5/SHA1/SHA256) при запросе.

Скрипт выведет результаты сканирования и сохранит полный ответ в virustotal_response.json.

Вывод результатов
Скрипт отображает:

количество обнаруженных вредоносных, подозрительных, безопасных и не определённых файлов;

информацию о файле (размер, тип, имя);

правила фильтрации (если есть);

полный JSON‑ответ сохраняется в файл для детального анализа.

Устранение неполадок
Ошибка «API‑ключ не найден»: убедитесь, что переменная VT_API_KEY установлена правильно.

Файл не найден в базе VirusTotal: проверьте правильность хеша или загрузите файл на VirusTotal для сканирования.
